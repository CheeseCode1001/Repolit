# 💊 Developer Painkiller

**Developer Painkiller** is an AI-powered engine designed to eliminate the "onboarding headache." By leveraging Gemini 1.5 Pro's massive context window, it reads entire repositories and explains them to you like a lead architect would.

## 🚀 Features
* **Instant Context:** Get a high-level summary of any repo in seconds.
* **Visual Logic:** Auto-generated Mermaid.js diagrams showing how data flows.
* **Automated Onboarding:** tailored guides for new contributors.
* **Security Pulse:** Quick scans for exposed keys and common vulnerabilities.

## 🛠 Tech Stack
* **Engine:** LangGraph + Google Gemini 1.5 Pro
* **Backend:** Python
* **Frontend:** Streamlit / Replit Deploy
* **Visuals:** Mermaid.js

## Development Setup
1. Clone your fork.
2. Install dependencies: `pip install -r requirements.txt`.
3. Set up your `.env` file with `GEMINI_API_KEY` and `GITHUB_CLIENT_ID`.
